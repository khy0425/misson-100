📋 개인정보처리방침

💀 만삣삐들아! 너희의 개인정보는 ALPHA EMPEROR가 철저히 보호한다! 💀

1. 개인정보의 처리 목적

Mission 100 앱은 다음의 목적을 위하여 개인정보를 처리합니다:

  • 🏋️‍♂️ 사용자의 운동 기록 및 진행 상황 관리
  • 📊 개인 맞춤형 운동 프로그램 제공
  • 🏆 업적 및 레벨 시스템 운영
  • 📱 앱 기능 개선 및 사용자 경험 향상
  • 🔔 운동 알림 및 동기부여 메시지 제공

2. 처리하는 개인정보의 항목

Mission 100 앱에서 처리하는 개인정보는 다음과 같습니다:

  • 필수정보: 사용자 닉네임, 운동 기록, 앱 사용 기록
  • 선택정보: 알림 설정, 테마 설정, 백업 데이터
  • 자동수집정보: 앱 사용 통계, 오류 로그 (개인식별 불가능한 형태)

3. 개인정보의 처리 및 보유 기간

  • 사용자가 앱을 삭제하거나 데이터 삭제를 요청할 때까지 보유
  • 법령에 따른 보존 의무가 있는 경우 해당 기간까지 보유
  • 사용자는 언제든지 설정에서 데이터 삭제를 요청할 수 있습니다

4. 개인정보의 제3자 제공

🛡️ Mission 100은 사용자의 개인정보를 제3자에게 제공하지 않습니다!

단, 다음의 경우는 예외입니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법령의 규정에 의한 경우
  • 수사기관의 수사목적으로 법령에 정해진 절차에 따라 요구받은 경우

5. 개인정보 처리의 위탁

Mission 100은 서비스 향상을 위해 다음과 같이 개인정보 처리를 위탁하고 있습니다:

  • Google Firebase: 데이터 저장 및 분석
  • Google AdMob: 광고 서비스 제공

6. 정보주체의 권리·의무 및 행사방법

사용자는 다음과 같은 권리를 가집니다:

  • 개인정보 처리현황 통지 요구
  • 개인정보 열람 요구
  • 개인정보 정정·삭제 요구
  • 개인정보 처리정지 요구

7. 개인정보의 안전성 확보조치

  • 🔐 데이터 암호화 저장
  • 🛡️ 접근권한 관리 및 제한
  • 📱 로컬 데이터 보호
  • 🔄 정기적인 보안 점검

📋 Privacy Policy

💀 ALPHA EMPEROR protects your personal information thoroughly! 💀

1. Purpose of Processing Personal Information

Mission 100 app processes personal information for the following purposes:

  • 🏋️‍♂️ Managing user's workout records and progress
  • 📊 Providing personalized workout programs
  • 🏆 Operating achievement and level systems
  • 📱 Improving app functionality and user experience
  • 🔔 Providing workout notifications and motivational messages

2. Items of Personal Information Processed

Personal information processed by Mission 100 app includes:

  • Required Information: User nickname, workout records, app usage records
  • Optional Information: Notification settings, theme settings, backup data
  • Automatically Collected: App usage statistics, error logs (in non-identifiable form)

3. Processing and Retention Period

  • Retained until user deletes the app or requests data deletion
  • Retained for the required period if there are legal obligations
  • Users can request data deletion anytime through settings

4. Provision of Personal Information to Third Parties

🛡️ Mission 100 does not provide personal information to third parties!

Exceptions include:

  • When explicit user consent is given
  • When required by law
  • When requested by investigative agencies according to legal procedures

5. Consignment of Personal Information Processing

Mission 100 consigns personal information processing as follows for service improvement:

  • Google Firebase: Data storage and analytics
  • Google AdMob: Advertisement service provision

6. Rights and Obligations of Data Subjects

Users have the following rights:

  • Request notification of personal information processing status
  • Request access to personal information
  • Request correction/deletion of personal information
  • Request suspension of personal information processing

7. Security Measures for Personal Information

  • 🔐 Encrypted data storage
  • 🛡️ Access control and restrictions
  • 📱 Local data protection
  • 🔄 Regular security inspections
