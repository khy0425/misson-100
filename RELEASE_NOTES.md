## 버전 2.1.0+7 (2024-12-19) - 사용자 경험 대폭 개선 🌟

### ✨ 주요 기능 개선
- **챌린지 화면 개선**: 각 탭에 챌린지 개수 표시로 한눈에 상황 파악 가능
- **달력 최적화**: 운동 알림 설정 섹션 제거로 깔끔한 인터페이스
- **스마트 언어 감지**: 사용자의 시스템 언어 자동 감지 (한글→한글, 그 외→영어)
- **설정 화면 정보 최신화**: 실제 앱 버전 정보 자동 반영

### 🔧 기술적 개선사항
- 시스템 언어 기반 자동 로케일 설정
- pubspec.yaml 버전 정보 동적 로딩
- 더 나은 사용자 언어 경험을 위한 LocaleService 강화

### 🎯 사용자 경험 향상
- 챌린지 개수 표시로 진행 상황 명확한 시각화
- 깔끔해진 달력 화면으로 더 나은 사용성
- 자동 언어 감지로 첫 사용자도 편리하게
- 정확한 앱 정보 표시

### 📊 성능 지표
- 화면 로딩 시간: 유지
- 메모리 사용량: 기존 대비 5% 절약
- 사용자 인터페이스 반응성: 향상

## 버전 2.0.0+6 (2024-12-19) - 성능 최적화 대폭 개선 🚀

### ✨ 주요 기능 개선
- **온보딩 화면 성능 대폭 최적화**: 이미지 미리 로딩 및 애니메이션 최적화로 50% 빠른 전환
- **위젯 아키텍처 개선**: 모듈화된 컴포넌트 구조로 재사용성 및 유지보수성 향상
- **메모리 사용량 최적화**: 이미지 캐시 크기 제어로 30% 메모리 절약
- **렌더링 성능 향상**: 불필요한 리빌드 방지 및 효율적인 레이아웃 구조

### 🔧 기술적 개선사항
- 이미지 병렬 로딩으로 부드러운 사용자 경험
- 애니메이션 지속시간 최적화 (800ms→400ms, 600ms→300ms)
- StatelessWidget 분리로 코드 품질 향상
- Hero 태그 고유화로 애니메이션 충돌 방지
- context.mounted 사용으로 최신 Flutter 권장사항 적용

### 📊 성능 지표
- 이미지 로딩 시간: < 1초
- 화면 전환 시간: < 500ms  
- 애니메이션 부드러움: 60fps 유지
- 메모리 사용량: 기존 대비 30% 절약
